# 依赖
## install Vuex
`npm i vuex@3 `
## install axios
`npm i vue-axios`
## install router
`npm i vue-router@3`
## install less
`npm i less less-loader@7.3.0`
## install mockjs mock数据
`npm i mockjs`
## install nprogress 加载进度条插件
`npm i nprogress`
## install swiper 轮播图
`npm i swiper@5`
## install uuid 
`npm i uuid`
## install animate.css 动画效果
`npm i animate.css`
